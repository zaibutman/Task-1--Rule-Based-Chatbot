# 🤖 Rule-Based AI Chatbot

A simple **Rule-Based AI Chatbot** developed in **Python** as part of **Project 1** during my **Artificial Intelligence Internship at Decode Labs**.

The chatbot interacts with users through predefined rules, demonstrating the fundamentals of Artificial Intelligence, decision-making logic, and control flow using Python.

---

## 📌 Project Overview

This chatbot is designed to simulate a basic conversation by matching user inputs with predefined responses stored in a Python dictionary. It continuously interacts with the user until an exit command is given.

Unlike modern AI chatbots powered by Machine Learning or Large Language Models (LLMs), this chatbot is entirely **rule-based**, making it an excellent introduction to the core concepts of conversational AI.

## ✨ Features

- 🤖 Interactive command-line chatbot
- 🔁 Continuous conversation using `while True`
- 📚 Dictionary-based knowledge base
- 🔍 Input normalization using `.lower()` and `.strip()`
- 💬 Multiple predefined responses
- 🚪 Supports multiple exit commands (`bye`, `exit`, `quit`)
- ❓ Gracefully handles unknown inputs
- 📝 Clean, readable, and beginner-friendly code
